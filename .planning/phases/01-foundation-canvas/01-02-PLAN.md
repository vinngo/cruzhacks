  <action>
    Create app/template.tsx to enable smooth page transitions (entry and exit animations).

    IMPORTANT: Use template.tsx, NOT layout.tsx, for transitions. Templates re-render on navigation, enabling exit animations. Layouts do not re-render, so exit animations won't work.

    Create app/template.tsx:
    - Mark as 'use client' (uses Framer Motion)
    - Import motion from 'framer-motion'
    - Wrap children in motion.div with animations:
      - initial: opacity 0, y: 20
      - animate: opacity 1, y: 0
      - exit: opacity 0, y: -20
      - transition: duration 0.3 seconds
    - Export default function Template({ children })

    This provides:
    - Fade in + slide up on page entry
    - Fade out + slide down on page exit
    - 300ms duration for smooth but not sluggish feel

    Test by navigating landing -> workspace (even if workspace is 404, transition should show).
  </action>
  <verify>
    Run `npm run dev` and test navigation:

  </verify>
